## :toolbox: Getting Started

### :bangbang: Prerequisites

### :running: Run Locally

Clone the project

```bash
https://github.com/TakhirKudusov/arkana-gaming-landing-nextjs
```
Install dependencies
```bash
npm i
```
Build the project
```bash
npm run build
```
Start the project
```bash
npm run start
```
